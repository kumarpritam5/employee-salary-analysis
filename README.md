# 📊 Employee Salary Analysis

A web-based application to analyze employee salary data using **JavaScript, SQL-like queries, and data visualization**.

---

## 🚀 Overview

This project allows users to upload employee datasets, run queries, and visualize insights through charts. It is designed to simulate real-world **data analysis workflows** in a simple and interactive way.

## ⚙️ Installation & Setup
Clone the repository:
```
git clone https://github.com/kumarpritam5/employee-salary-analysis.git
cd employee-salary-analysis
```
Install dependencies:
```
npm install
```
Run the project:
```
npm start
```
Open in browser:
```
http://localhost:3000
```
